# DataOps Demo Guide

## 5-Minute Demo

1. Run the cleaning, KPI, and dashboard scripts.
2. Open `output_samples/dashboard_preview.html`.
3. Open `workflows/n8n_project_workflow.json` and point out scheduled Postgres read, KPI calculation, OpenRouter insights, KPI storage, and Gmail delivery.
4. Open `database/schema.sql` and show `sales_orders_clean`, `kpi_reports`, and `power_bi_weekly_kpis`.
