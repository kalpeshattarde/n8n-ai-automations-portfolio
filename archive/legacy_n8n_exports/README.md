# Legacy n8n Exports

This folder contains earlier live-integration n8n workflow exports:

- `akceleratehq-free-audit-auto-proposal`
- `akceleratehq-whatsapp-chatbot`
- `coaching-owner-ai-assistant`

They are retained as reference examples of credentialed n8n builds. The main recruiter-facing portfolio is now the root-level three-project system because it is public-safe, locally runnable, and easier to evaluate without private API access.

Before using any legacy workflow in production, review credentials, rotate exposed tokens if any were ever used in testing, add explicit failure branches, and test in a sandbox n8n instance.
