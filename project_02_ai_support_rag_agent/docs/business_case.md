# Business Case

Support teams waste time answering repeat questions, but risky tickets still need human judgment. This project demonstrates a support RAG workflow that automates draft creation while escalating sensitive or low-confidence tickets.

## Business Value

- Lower first-response time for repeat support questions.
- Consistent customer replies grounded in the knowledge base.
- Escalation for refunds, billing disputes, complaints, SSO, and technical incidents.
- Analytics report showing categories, approval volume, and escalation drivers.

## ROI Model

The simulation estimates saved handling minutes for tickets routed to approval with high confidence. In production, this should be measured against first-response SLA, resolution time, deflection rate, and customer satisfaction.
