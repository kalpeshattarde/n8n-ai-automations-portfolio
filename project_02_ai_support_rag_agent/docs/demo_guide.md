# Support RAG Demo Guide

## 5-Minute Demo

1. Run the retrieval and triage scripts.
2. Open `output_samples/escalation_report_sample.md`.
3. Open `workflows/n8n_project_workflow.json` and point out Gmail intake, PostgreSQL ticket storage, KB retrieval, OpenRouter drafting, approval, and escalation.
4. Open `database/schema.sql` and show tickets, KB articles, responses, approvals, and errors.
