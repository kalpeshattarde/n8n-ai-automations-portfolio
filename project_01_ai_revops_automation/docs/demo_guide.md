# RevOps Demo Guide

## 5-Minute Demo

1. Run the local scoring scripts.
2. Open `output_samples/daily_owner_report_sample.md`.
3. Open `workflows/n8n_project_workflow.json` and point out Google Sheets intake, OpenRouter scoring, PostgreSQL CRM storage, Gmail approval, Calendar hold, and error logging.
4. Open `database/schema.sql` and show `crm_leads`, `lead_events`, `approval_tasks`, and `workflow_errors`.
